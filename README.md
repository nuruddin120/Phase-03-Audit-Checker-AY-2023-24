# অডিট সিলেকশন চেকার ২০২৩-২০২৪

আয়কর বিভাগের অডিট নির্বাচন তালিকা (২০২৩-২০২৪ কর বর্ষ) থেকে TIN নম্বর দিয়ে সার্চ করুন।

## ফিচারস

- TIN নম্বর দিয়ে ইনস্ট্যান্ট সার্চ
- অডিটে নির্বাচিত হলে ক্রমিক নং, Zone, Circle, Submission Type ও Assessment Year দেখায়
- সম্পূর্ণ অফলাইন — কোনো সার্ভার বা ডাটাবেজ লাগে না
- মোবাইল রেসপন্সিভ ডিজাইন
- মোট ৫,০১৪ টি TIN রেকর্ড

## ডেপ্লয়মেন্ট

### Vercel-এ ডেপ্লয় করুন

1. এই রিপোজিটরি GitHub-এ পুশ করুন
2. [vercel.com](https://vercel.com)-এ যান এবং GitHub দিয়ে লগইন করুন
3. "Add New Project" → আপনার রিপো সিলেক্ট করুন
4. কোনো সেটিংস পরিবর্তন না করে "Deploy" চাপুন
5. আপনার সাইট লাইভ!

## টেকনোলজি

- Pure HTML, CSS, JavaScript
- কোনো ফ্রেমওয়ার্ক বা বিল্ড স্টেপ নেই
- ডেটা JSON হিসেবে HTML-এর ভিতরে এমবেডেড
